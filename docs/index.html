pydeepcausalml

Latest release: 0.2.0 · Requires: Python ≥ 3.9 · License: MIT

Deep learning models for causal inference in Python with Pytorch: treatment-effect estimation, causal structure learning, temporal causal discovery, and counterfactual forecasting — behind one consistent, sklearn-style API.

pip install pydeepcausalml
Package PyPI · GitHub
Docs Tutorial · Glossary
Author Zia U. Ahmed · Upatta Analytics

What is it?

PyDeepCausalML consolidates deep causal models from the Deep Causal ML tutorial series into a tested, installable Python package.

Use it when you need to:

  • Estimate average or individual treatment effects from observational data
  • Discover causal graphs from static or time-series data
  • Run counterfactual and interventional queries with neural structural models
  • Train on CPU, CUDA, or Apple MPS with the same sklearn-style API

Installation

pip install pydeepcausalml            # core
pip install "pydeepcausalml[plot]"    # + matplotlib/seaborn plotting helpers
pip install "pydeepcausalml[dev]"     # + test and lint tooling

From source:

git clone https://github.com/zia207/PyDeepCausalML && cd PyDeepCausalML
pip install -e ".[dev]"
pytest

Main features

  • Treatment effects: TARNet, CFRNet, DragonNet, NeuralDML, GANITE, CEVAE
  • Generative causal models: CausalVAE, IVAE, CausalDiscrepancyVAE, CausalGAN, DSCM, CausalEGM
  • Structure learning: NOTEARS, DAGGNN, DagmaLinear, GranDag, CASTLE, DynoTEARS
  • Time series: neural Granger (cMLP, cLSTM), TCDF, CausalTransformer, GVAR, CausalGNN
  • Counterfactuals: DeepSynth, CRN, GNet, CausalLSTMForecaster, DeepSCM, DECI
  • Utilities: synthetic datasets with ground truth, PEHE/SHD metrics, causal graph plotting

Quickstart

from pydeepcausalml import DragonNet, NeuralDML
from pydeepcausalml.datasets import make_confounded_data
from pydeepcausalml.metrics import pehe

df = make_confounded_data(n=5000, random_state=0)
X = df[["age", "education", "prior_income"]].values
t, y = df["treatment"].values, df["outcome"].values

est = DragonNet(epochs=200, random_state=0).fit(X, t, y)
print("ATE:", est.predict_ate(X))
print("PEHE:", pehe(df["true_cate"], est.predict_cate(X)))

dml = NeuralDML(n_splits=2, random_state=0).fit(X, t, y)
print("DML ATE:", dml.predict_ate(), "95% CI:", dml.confidence_interval())
